🚀 КАК ЗАГРУЗИТЬ НА ХОС (САМЫЙ ПРОСТОЙ СПОСОБ):

1. СОЗДАЙТЕ GitHub АККАУНТ:
   → Перейдите на https://github.com
   → Нажмите "Sign up"
   → Создайте аккаунт

2. СОЗДАЙТЕ НОВЫЙ РЕПОЗИТОРИЙ:
   → Перейдите на https://github.com/new
   → Имя: zorssms
   → Описание: Real-time messenger
   → Выберите Public
   → Нажмите "Create repository"

3. ЗАГРУЗИТЕ ФАЙЛЫ (БЕЗ GIT!):
   → На странице репозитория нажмите "Add file"
   → Выберите "Upload files"
   → Перейдите в папку: c:\Users\PC\Desktop\ортиртимт\zorssms
   → Загрузите ВСЕ файлы КРОМЕ папки node_modules:
     - index.html
     - server.js
     - package.json
     - package-lock.json
     - Procfile
     - .gitignore
     - .env.example
     - README.md
     - DEPLOY.md
     - UPLOAD_GUIDE.md
     - INSTALL.md
     - CHECKLIST.md
     - css/ (вся папка)
     - js/ (вся папка)
   → Нажмите "Commit changes"

4. РАЗВЕРНИТЕ НА RENDER.COM:
   → Перейдите на https://render.com
   → Создайте аккаунт (можно через GitHub)
   → Нажмите "+ New" → "Web Service"
   → Нажмите "Connect account" (GitHub)
   → Выберите репозиторий "zorssms"
   → Заполните:
     - Name: zorssms
     - Runtime: Node
     - Build Command: npm install
     - Start Command: npm start
   → Нажмите "Deploy"
   → Дождитесь "Your service is live ✓"

5. ИСПОЛЬЗУЙТЕ МЕССЕНДЖЕР:
   → Откройте URL (будет выдан Render.com)
   → Нажмите "Зарегистрироваться"
   → Система создаст ваш ID (USER_XXXXXXXXX)
   → Поделитесь ID с друзьями!
